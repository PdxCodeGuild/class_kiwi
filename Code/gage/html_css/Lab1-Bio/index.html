Arthur Morgan

Arthur Morgan is the main outlaw character in Rockstars 2018 video game "Red Dead Redemption 2". Born in 1863, Author Morgan had a rough up bringing in Cincinnati, Ohio and after losing both of his parents he sooned joined Dutch van der Linde's gang in 1877. The rest is history.
